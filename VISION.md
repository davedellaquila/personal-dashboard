## Design Principles

### 1. Zero-Friction Capture
**Rae Assist Principle:** Quick input without fighting the system  
**Modern Execution:** 
- Voice â†’ text in seconds
- Browser extension â†’ one-click save
- Cmd+K â†’ instant note
- SMS/email â†’ forward to knowledge base

**Success Metric:** Add new knowledge in <10 seconds

### 2. Intelligent Connections
**Rae Assist Principle:** Automatic linking between related items  
**Modern Execution:**
- AI embeddings for semantic similarity
- Topic extraction and auto-tagging
- Suggested connections (with confidence scores)
- User can confirm/edit/remove links

**Success Metric:** Discover 1+ unexpected connection per week

### 3. Multiple Views
**Rae Assist Principle:** Seamless navigation between related items  
**Modern Execution:**
- Search (find specific thing fast)
- Graph (explore connections visually)
- Timeline (when did I learn this?)
- Tags/Topics (browse by category)
- Resurface (what should I review?)

**Success Metric:** Find any item in <30 seconds

### 4. Future-Proof Storage
**Rae Assist Challenge:** Proprietary database, migration difficulties  
**Modern Solution:**
- Plain markdown files (human-readable)
- Open formats (JSON, CSV export)
- Local-first (works offline)
- Cloud-optional (sync when you want)

**Success Metric:** Full export in open format, any time

### 5. Progressive Enhancement
**Start Simple â†’ Add Power Features**

**Phase 1:** File storage + basic search (works immediately)  
**Phase 2:** AI embeddings + semantic search (more powerful)  
**Phase 3:** Graph view + resurface (advanced features)

**Success Metric:** Useful from day 1, better every week
